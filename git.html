Google It!

Description:

  • The name is self explanatory and yes, this event is introduced with the notion of testing the surfing knowledge and ability of the students.
  • It is a team event which allows 2 members.
  • The event consists of two rounds.

Round 1:

  • In this round the student has to search for the given google doodle and state them.
  • This is an elimination round.

Round 2:

  • The selected team has to search for the surprise element provided during the round.

Rules:

  • Mobile is strictly prohibited inside the event.
  • The team should not exceed the given time else disqualified.
  • Event Organizers decision are final.
